SSH 접속 (Windows → VM)

ssh ccc@<VM_IP> -p 2202   # web
ssh ccc@<VM_IP> -p 2203   # siem
ssh ccc@<VM_IP> -p 2204   # bastion
ssh ccc@<VM_IP> -p 2205   # attacker

기본 암호는 .env의 CCC_SSH_PASS (기본값 ccc).
