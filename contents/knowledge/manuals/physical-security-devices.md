# 사이버보안 물리 기자재 가이드

> CCC 교육에 활용 가능한 물리 보안 도구 종합 카탈로그

---

## 1. 무선/RF 해킹

### Flipper Zero
- **용도**: RFID/NFC 복제, Sub-GHz 신호 캡처/재전송, IR 리모트, iButton, GPIO
- **가격**: ~20만원
- **실습 예시**:
  - 125kHz RFID 출입카드 복제
  - 차량 키리스 엔트리 신호 캡처 (Sub-GHz)
  - TV/에어컨 IR 신호 학습+재전송
  - NFC 카드 UID 에뮬레이션
  - BadUSB 모드 (키스트로크 인젝션)
- **교과목**: 공격 심화, 물리보안, IoT 보안

### HackRF One (SDR)
- **용도**: 1MHz~6GHz 전 대역 수신/송신, 무선 프로토콜 분석
- **가격**: ~40만원
- **실습 예시**:
  - FM 라디오 수신/송신
  - ADS-B 항공기 위치 추적
  - GPS 스푸핑 시연 (차폐 환경)
  - 블루투스 스니핑
  - LoRa/Zigbee IoT 프로토콜 분석
  - 무선 도어벨/센서 신호 분석
- **교과목**: 공격 심화, IoT 보안, 자율보안

### RTL-SDR (수신 전용)
- **용도**: 25MHz~1.75GHz 수신, 저비용 SDR 입문
- **가격**: ~3만원
- **실습 예시**:
  - ADS-B 항공기 추적
  - 기상 위성 이미지 수신
  - 무선 호출기/페이저 수신
  - FM/AM 라디오 분석
- **교과목**: 입문용, IoT 보안

### WiFi Pineapple
- **용도**: 무선 AP 위장(Evil Twin), MITM, 크레덴셜 캡처
- **가격**: ~15만원
- **실습 예시**:
  - Rogue AP 생성 → 접속 유도
  - HTTPS 다운그레이드 공격 시연
  - DNS 스푸핑
  - 캡티브 포털 피싱
- **교과목**: 공격, 네트워크 보안

### WiFi 어댑터 (모니터 모드)
- **추천**: Alfa AWUS036ACH, AWUS036AXML
- **가격**: 3~7만원
- **실습 예시**:
  - WPA2 핸드셰이크 캡처 (airodump-ng)
  - WPA2 크래킹 (hashcat/aircrack-ng)
  - 디인증 공격 (aireplay-ng)
  - 숨겨진 SSID 탐지
  - 무선 클라이언트 추적
- **교과목**: 공격, 네트워크 보안

## 2. RFID/NFC/카드 보안

### Proxmark3 RDV4
- **용도**: 125kHz/13.56MHz RFID/NFC 읽기/쓰기/에뮬레이션/크래킹
- **가격**: ~30만원
- **실습 예시**:
  - Mifare Classic 키 크래킹 (darkside/nested attack)
  - HID iClass 카드 복제
  - EM4100 카드 복제
  - 은행카드 NFC 데이터 읽기 (비접촉 결제)
  - 사원증/학생증 RFID 분석
  - 커스텀 카드 프로그래밍
- **교과목**: 물리보안, 컴플라이언스

### ACR122U (NFC 리더)
- **용도**: 13.56MHz NFC 읽기/쓰기, 저비용 NFC 실습
- **가격**: ~5만원
- **실습 예시**:
  - NFC 태그 읽기/쓰기
  - NDEF 메시지 조작
  - 교통카드 잔액 읽기
- **교과목**: 입문용

## 3. USB/HID 공격

### USB Rubber Ducky
- **용도**: 키스트로크 인젝션, 자동 페이로드 실행
- **가격**: ~7만원
- **실습 예시**:
  - 10초 만에 리버스 셸 설치
  - WiFi 비밀번호 탈취 스크립트
  - 자격 증명 수집 (mimikatz 자동 실행)
  - 시스템 정보 수집 + 외부 전송
  - 랜섬웨어 시뮬레이션 (암호화 시연)
- **교과목**: 공격, 물리보안, 보안인식

### Bash Bunny
- **용도**: USB Rubber Ducky 상위 호환, 이더넷/스토리지 에뮬레이션
- **가격**: ~15만원
- **실습 예시**:
  - USB 이더넷 에뮬레이션 → MITM
  - 네트워크 트래픽 캡처
  - SMB 릴레이 공격
  - Exfiltration via USB 스토리지 에뮬레이션
- **교과목**: 공격 심화, Red Team

### O.MG Cable
- **용도**: 정상 USB 케이블로 위장한 키로거/페이로드 실행기
- **가격**: ~15만원
- **실습 예시**:
  - 키로깅 (타이핑 내용 WiFi로 전송)
  - 지오펜싱 (GPS 기반 자동 실행)
  - 원격 제어 (WiFi AP 내장)
- **교과목**: 공격 심화, 보안인식

### USB Killer
- **용도**: USB 포트를 통한 전기적 파괴 (교육 목적 시연만)
- **가격**: ~10만원
- **실습**: 폐기 장비에서 시연 → USB 포트 보호 필요성 교육
- **교과목**: 물리보안, 컴플라이언스

## 4. 네트워크 장비

### 관리형 L2/L3 스위치
- **추천**: Cisco Catalyst 1000, TP-Link TL-SG3428
- **가격**: 15~50만원
- **실습 예시**:
  - VLAN 구성 + 트렁킹
  - 포트 미러링 → IDS 연동
  - 802.1X 포트 인증
  - ARP 스푸핑 방지 (DAI)
  - DHCP Snooping
  - STP 공격 방어
- **교과목**: 보안운영, 네트워크 보안

### LAN Turtle
- **용도**: 이더넷 포트에 삽입하는 침투 장치
- **가격**: ~8만원
- **실습 예시**:
  - 네트워크 정찰 (내부에서)
  - DNS 스푸핑
  - VPN 리버스 터널
  - 트래픽 캡처
- **교과목**: Red Team, 공격 심화

### Packet Squirrel
- **용도**: 인라인 패킷 캡처/수정 장치
- **가격**: ~8만원
- **실습**: MITM, 패킷 로깅, VPN 리다이렉트
- **교과목**: 네트워크 보안

### Shark Jack
- **용도**: 이더넷 포트에 꽂으면 자동 정찰+페이로드 실행
- **가격**: ~8만원
- **실습**: 물리적 접근 시 네트워크 정찰 자동화
- **교과목**: Red Team

## 5. 하드웨어 해킹/IoT

### Raspberry Pi 5
- **가격**: ~10만원
- **실습 예시**:
  - 취약한 IoT 서버 구축 (의도적 취약점)
  - 허니팟 운영 (Cowrie, Dionaea)
  - 네트워크 센서/모니터
  - Pi-hole (DNS 기반 광고/악성코드 차단)
  - 무선 AP 구축 (hostapd)
- **교과목**: 전 과목

### Arduino + 센서
- **가격**: ~3만원
- **실습**: 물리적 센서 데이터 → 보안 모니터링 연동
- **교과목**: IoT 보안, 자율보안

### Bus Pirate
- **용도**: UART, SPI, I2C, JTAG 프로토콜 분석
- **가격**: ~5만원
- **실습 예시**:
  - IoT 디바이스 UART 콘솔 접속
  - 펌웨어 덤프 (SPI Flash)
  - JTAG 디버깅
  - EEPROM 읽기/쓰기
- **교과목**: IoT 보안, 하드웨어 해킹

### JTAGulator
- **용도**: JTAG/UART 핀 자동 탐지
- **가격**: ~15만원
- **실습**: 알려지지 않은 PCB에서 디버그 포트 찾기
- **교과목**: 하드웨어 해킹

### Logic Analyzer (Saleae Logic)
- **가격**: 5~40만원
- **실습**: 디지털 신호 캡처 + 프로토콜 디코딩
- **교과목**: IoT 보안

## 8. 드론/자율주행/로봇 보안

### 교육용 드론
#### Tello EDU (DJI)
- **가격**: ~15만원
- **실습 예시**:
  - WiFi 프로토콜 분석 (드론↔컨트롤러)
  - 드론 명령어 인젝션 (UDP 패킷 조작)
  - 드론 하이재킹 (deauth + 재연결)
  - 비행 경로 GPS 스푸핑 (시뮬레이션)
  - 영상 스트림 가로채기 (RTSP)
  - 펌웨어 추출 + 분석
- **교과목**: IoT 보안, 공격 심화

#### CoDrone EDU (Robolink)
- **가격**: ~20만원
- **실습**: Python으로 제어 → API 취약점 분석, 명령어 인젝션
- **교과목**: AI 보안 에이전트, 자율보안

#### 드론 탐지/방어 키트
- **WiFi 어댑터 + Python 스크립트**로 드론 WiFi 신호 탐지
- **SDR(HackRF)**로 드론 RF 주파수 분석
- **실습**: 드론 침입 탐지 시스템 구축
- **교과목**: 방어 운영, SOC

### 자율주행 키트

#### NVIDIA JetBot (Jetson Nano 기반)
- **가격**: ~30만원
- **실습 예시**:
  - 카메라 센서 데이터 변조 (adversarial patch)
  - LiDAR/초음파 센서 스푸핑
  - 자율주행 AI 모델 공격 (적대적 입력)
  - V2X 통신 MITM
  - OTA 업데이트 변조
  - 원격 제어 하이재킹
- **교과목**: AI Safety, AI 보안, 자율보안

#### Donkey Car (오픈소스 자율주행)
- **가격**: ~20만원 (Raspberry Pi + 차체 + 카메라)
- **실습 예시**:
  - 자율주행 AI 모델 포이즈닝 (학습 데이터 변조)
  - 정지 표지판에 스티커 → 인식 실패 유도
  - 통신 채널 가로채기
  - 원격 비상 정지 메커니즘 우회
- **교과목**: AI Safety, 자율보안

#### AWS DeepRacer
- **가격**: ~40만원
- **실습**: 강화학습 모델 공격, 보상 함수 조작
- **교과목**: AI 보안, 자율보안
